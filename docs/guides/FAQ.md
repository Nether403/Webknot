# Frequently Asked Questions (FAQ)

Quick answers to common questions about using LovaBolt.

---

## 🚀 Getting Started

### What is LovaBolt?
LovaBolt is a wizard that helps you create detailed website descriptions (prompts) for AI development tools like Bolt.new and Lovable.dev. It guides you through every aspect of your website design so AI can build exactly what you envision.

### Do I need coding experience?
No! LovaBolt is designed for everyone - designers, business owners, content creators, and anyone with a website idea. No technical knowledge required.

### Is LovaBolt free?
Yes! LovaBolt is completely free to use.

### Which browsers work with LovaBolt?
LovaBolt works best on:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### How long does it take to create a prompt?
Most users complete the wizard in 5-10 minutes. Take your time to get the best results!

---

## 💾 Saving & Progress

### Does LovaBolt save my work?
Yes! Your progress is automatically saved every second to your browser's local storage. You can close the tab and come back anytime.

### Can I work on multiple projects?
Currently, LovaBolt saves one project at a time in your browser. To start a new project, you'll need to generate and save your current prompt first.

### What if I accidentally close the browser?
No worries! Your work is saved automatically. Just reopen LovaBolt and you'll see your progress.

### Can I export my project?
You can copy your generated prompt and save it anywhere (Google Docs, Notion, text file). Project export/import feature is coming soon!

### Will my project be saved if I use a different browser?
No, projects are saved per browser. Use the same browser to access your saved work.

## 🎨 Using the Wizard

### Can I skip steps?
You can navigate between steps freely using the sidebar, but we recommend completing all steps for the best prompt quality.

### Can I go back and change things?
Absolutely! Click any step in the sidebar or use the "Back" button to make changes anytime.

### What if I make a mistake?
Use Ctrl/Cmd + Z to undo recent changes, or simply go back to the step and change your selection.

### How do I know which options to choose?
- Use **Smart Defaults** for intelligent suggestions
- Check **AI Suggestions** panel for compatible options
- Look at the **Harmony Score** to ensure choices work together
- Read option descriptions for guidance

### What's the difference between Detailed and Basic prompts?
- **Detailed Prompt**: Complete specifications with all details (recommended for best results)
- **Basic Prompt**: Concise summary for simple projects

## 🤖 AI Features

### What is Smart Defaults?
Smart Defaults automatically pre-selects options based on your project type. For example, choosing "Portfolio" suggests Minimalist style, clean layouts, and appropriate components.

### How accurate is the Prompt Quality Score?
Very accurate! It analyzes your prompt for completeness, best practices, and potential issues. Aim for 85+ for excellent results.

### What does the Harmony Score mean?
The Harmony Score (0-100) shows how well your design choices work together:
- **90-100**: Excellent harmony
- **75-89**: Good combination
- **60-74**: Minor issues
- **Below 60**: Consider adjustments

### Should I fix all warnings?
- **Red warnings**: Yes, fix these (important issues)
- **Yellow tips**: Recommended but optional
- **Green checks**: You're doing great!

### How does Natural Language Input work?
Just describe your project naturally (e.g., "I want a minimalist portfolio with blue colors") and LovaBolt detects:
- Project type
- Design style
- Color preferences
- And more!

### What are Context-Aware Suggestions?
AI recommendations that appear based on your selections. For example, choosing Glassmorphism style triggers suggestions for compatible colors and components.

## 🎨 Design Choices

### How many components should I choose?
Start with 3-5 essential components. Minimalist designs work best with fewer components (5-7), while complex sites can have more (8-12).

### Can I use my own brand colors?
Yes! Select "Custom" in the color theme step and enter your specific colors.

### What if I don't have brand colors?
Use one of our preset themes - they're professionally designed to work well together.

### Which design style should I choose?
- **Portfolio/Creative**: Minimalist or Glassmorphism
- **Business/Corporate**: Modern Corporate
- **E-commerce**: Material Design
- **Tech/Startup**: Glassmorphism or Futuristic
- **Bold/Edgy**: Digital Brutalism

### How many animations should I add?
Less is more! Choose 2-3 subtle animations. Too many can be distracting and slow down your site.

## 🔧 React-Bits Components

### What are React-Bits components?
Professional, pre-built UI components you can add to your website. Think of them as building blocks - carousels, cards, timelines, animations, etc.

### How many React-Bits components are available?
93 components across three categories:
- 31 Backgrounds
- 37 UI Components
- 25 Animations

### Do I need to install React-Bits separately?
No! The generated prompt includes installation commands. The AI tool will handle installation when building your site.

### Can I search for specific components?
Yes! Use the search box on Background, Components, and Animations steps. You can also filter by tags.

### What if I can't find a component?
Try different keywords or browse by category. If you still can't find it, it might not be available yet.

## 📊 Quality & Results

### What makes a good prompt?
A good prompt:
- Has a quality score of 85+
- Includes responsive design
- Specifies accessibility features
- Has clear project description
- Includes all necessary details

### Why is my quality score low?
Common reasons:
- Missing responsive design
- No accessibility features
- Vague descriptions
- Missing technical requirements

Click "Apply Recommendations" to auto-fix issues!

### How can I improve my prompt?
1. Be specific in descriptions
2. Add responsive design
3. Include accessibility features
4. Specify performance requirements
5. Fix all red warnings

### Will a higher score guarantee better results?
A higher score means your prompt is more complete and detailed, which typically leads to better AI-generated websites. However, the AI tool's capabilities also matter.

## 🔄 Using Generated Prompts

### Which AI tool should I use?
- **Bolt.new**: Great for quick prototypes and full-stack apps
- **Lovable.dev**: Excellent for polished, production-ready sites
- **Claude Artifacts**: Good for custom implementations

### How do I use my prompt with Bolt.new?
1. Generate your prompt in LovaBolt
2. Click "Copy & Go to Bolt"
3. Paste in Bolt.new's input
4. Let AI build your website!

### How do I use my prompt with Lovable.dev?
1. Generate your prompt in LovaBolt
2. Click "Copy & Go to Lovable"
3. Paste in Lovable.dev's interface
4. Watch your site come to life!

### Can I edit the prompt before using it?
Yes! Copy it to a text editor, make changes, then paste into your AI tool.

### What if the AI doesn't understand my prompt?
LovaBolt prompts are optimized for AI tools, but if issues occur:
1. Check your quality score (aim for 85+)
2. Make sure you used the correct template
3. Try being more specific in descriptions
4. Regenerate with more details

## 🐛 Troubleshooting

### The wizard won't load
1. Refresh your browser (F5)
2. Clear browser cache
3. Try a different browser
4. Check if JavaScript is enabled

### My selections aren't saving
1. Don't use private/incognito mode
2. Enable cookies in browser settings
3. Use the same browser consistently
4. Try clicking another step to trigger save

### AI features aren't working
1. Complete earlier steps first
2. Wait a moment for AI to analyze
3. Refresh the page
4. Check internet connection

### Search isn't finding components
1. Try different keywords
2. Use tags instead
3. Browse by category
4. Check spelling

### Can't copy the prompt
1. Click "Copy" button again
2. Manually select and copy (Ctrl/Cmd + C)
3. Check clipboard permissions
4. Try a different browser

### Harmony Score shows warnings
1. Read the warning carefully
2. Click "Auto-Fix" if available
3. Manually adjust selections
4. Check AI suggestions panel

## ⌨️ Keyboard Shortcuts

### What keyboard shortcuts are available?
- `Ctrl/Cmd + →` - Next step
- `Ctrl/Cmd + ←` - Previous step
- `Ctrl/Cmd + G` - Generate prompt
- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Shift + Z` - Redo
- `Esc` - Close modal
- `Tab` - Navigate options
- `Enter` - Select option

### Why aren't shortcuts working?
1. Make sure no input field has focus
2. Check if modal is open (press Esc)
3. Verify browser doesn't override shortcuts
4. Click on the page first
